## Развёртывание

Для развёртывания, вам необходимо иметь:

- `node ^22`
- `yarn`
- `git`

```bash
git clone git@github.com:CSIT-legacy/csit-legacy-frontend
cd csit-legacy-frontend
yarn
yarn dev
```
